Arduino Keypad Interface using Proteus
*Project Overview

This project demonstrates interfacing a 4x4 Matrix Keypad with an Arduino Uno in Proteus Simulation. When a key is pressed, the corresponding value is displayed through the Serial Monitor.

*Components Used
1.Arduino Uno
2.4x4 Matrix Keypad
3.Virtual Terminal
4.Proteus 8 Professional

*Circuit Connections
Keypad Pin	Arduino Pin
Row A     	D7
Row B	      D6
Row C	      D5
Row D	      D4
Column 1	  D3
Column 2	  D2
Column 3	  D1
Column 4	  D0

*Features
~Reads keypad input
~Displays pressed key in Serial Monitor
~Easy Proteus simulation
~Suitable for password and security projects

*How to Run
1.Open the Arduino code in Arduino IDE.
2.Compile and generate HEX file.
3.Load HEX file into Arduino Uno in Proteus.
4.Run the simulation.
5.Press any key on the keypad.
6.Observe the output in the Virtual Terminal.
