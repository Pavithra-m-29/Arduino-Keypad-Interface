Arduino Keypad Interface using Proteus

This project demonstrates interfacing a 4x4 Matrix Keypad with Arduino Uno in Proteus Simulation.

Components Used

  • Arduino Uno
  • 4x4 Matrix Keypad
  • Virtual Terminal
  • Proteus 8 Professional

Features

  • Reads keypad input
  • Displays pressed key on Virtual Terminal
  • Proteus simulation support
  • Suitable for password-based projects
